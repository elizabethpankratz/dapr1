Research Design & Data

Semester 1 - Week 1

Exercises

You will be using simulated data for a 2x2 mixed design lexical decision task, in which 100 participants were required to decide whether a presented word was a real word or a non-word. The data include 100 rows (one for each participant) and 7 variables:

  • id: Participant ID
  • age: Age
  • language: Language group (1 = monolingual, 2 = bilingual)
  • rt_word: Reaction time (ms) for word trials
  • rt_nonword: Reaction time (ms) for non-word trials
  • acc_word: Accuracy for word trials
  • acc_nonword: Accuracy for non-word trials

Hints

Consider the following dataset on some of the most popular internet passwords, their strength, and how long it took for an algorithm to crack it. The data is available online at https://uoepsy.github.io/data/passworddata.csv.

Variable Name Description
rank Popularity in the database of released passwords
password Password
type Category of password
cracked Time to crack by online guessing
strength Strength = quality of password where 10 is highest, 1 is lowest
strength_cat Strength category (weak, medium, strong)
  1. Useful functions: the read_csv() from the tidyverse package or, alternatively, the read.csv() function from base R.

The data file is a CSV file (see the .csv in the file name extension). As such, you may wish to consider the tidyverse function read_csv()to read the data into R:

library(tidyverse)
pwords <- read_csv("https://uoepsy.github.io/data/passworddata.csv")
