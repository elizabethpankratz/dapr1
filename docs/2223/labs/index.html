Data Analysis for Psychology in R 1

Academic year 2022/23

Welcome

Welcome to the Data Analysis for Psychology in R 1 or, in short, DAPR1! This is a first year course introducing Psychology students to the fundamentals of data analysis and inferential thinking using the R programming language.

Each week, the course involves:

  • a lecture
  • a live R session
  • readings
  • a session

This webpage is only used to distribute the course materials. For information about course assessments and logistics, you should refer to the course LEARN page.
